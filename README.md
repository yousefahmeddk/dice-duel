🎲 Dice Challenge Game

A fun and simple Dice Game for two players!
Every time you refresh the page, both dice roll automatically — whoever gets the higher number wins 🎉.
If both dice show the same number, it’s a Draw!

🧩 Features

🎲 Two-player dice roll simulation

🔁 Auto-refresh to get new random results

🏆 Shows who wins or if it’s a draw

💻 Simple and responsive web design

🚀 How to Play

Open the website.

Refresh the page (or press F5).

Watch the dice roll automatically!

The winner will be shown at the top.
